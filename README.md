# ALX HTML & CSS Advanced Project

This project is part of the ALX HTML & CSS curriculum, focusing on advanced CSS techniques and styling. The goal is to create a fully styled webpage based on the provided Figma design.

## **Project Overview**

- **Objective**: To replicate the provided Figma design and implement advanced CSS styling techniques.
- **Tools**: HTML5, CSS3, Figma.
- **Features**:
  - Responsive design.
  - Custom typography and colors.
  - Use of Flexbox and CSS Grid for layout.
